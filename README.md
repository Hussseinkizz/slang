# slang

Functional programming library for TypeScript.

My experiment to learn more functional programming and other cool programming stuff from other languages as I try to implement them in TypeScript.

## API

- Result (Ok, Err)
- Maybe
- andThen
- Option
- Atom
- Expect
- Panic!
- Unwrap
- Zip, Unzip
- Try
- Catch
- Match
- Pipe
- Map
